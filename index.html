Мера / Mera — Tab Capper Firefox Extension · Browser Awaiting AMO approval · Available on GitHub 🦊 Firefox

Мера — measure, moderation. A Firefox extension with a single discipline: hold you to the number of tabs you have decided is enough. When you reach the limit, the browser tells you plainly — close one first. Pinned tabs are never counted. The rule is yours; Мера simply keeps it.
