Commitments

Accessibility & Motion

What this site will and will not do to you

Calling software contemplative is empty if the page announcing it flashes, pulses, and moves under the reader. These are the specific commitments, so they can be checked rather than taken on trust.

Motion

  • Nothing spins, flashes, strobes, or pulses. Everything that did — a rotating watermark, a flickering title, a pulsing button ring, a glowing heading — has been removed rather than slowed down.
  • All remaining motion animates opacity or transform only, so it stays smooth instead of juddering when the machine is busy.
  • Nothing moves faster than half a second, and the one decorative flourish — a slow sweep across the heading — happens once a minute.
  • prefers-reduced-motion is honoured everywhere, and it is the default for the quiet switch below.
  • No parallax on scroll unless the browser can drive it natively off a scroll timeline, where it cannot fall behind the scroll position.

The quiet switch

In the site header there is a toggle marked Motion. Turning it to Quiet stops every animation and transition on every page, hides the decorative flourishes entirely, and remembers the choice on this device. It is independent of your operating system, because no one should have to change a system preference to read a page in peace.

Reading

  • Every colour pair is checked against its background in both the light and dark themes.
  • Nothing conveys meaning by colour alone — category, status, and licence are all stated in words.
  • Text reflows and never scrolls sideways, down to a 320px screen.
  • Russian words carry lang="ru", so a screen reader pronounces them rather than spelling them out in English.
  • Every control is reachable and visible from the keyboard, with a focus ring that is never removed without a replacement.

The page itself

  • No trackers, no analytics, no cookies, no consent banner. Nothing about your visit is recorded anywhere.
  • No third-party subresources. Fonts, styles, scripts, and images are all served from this domain — no CDN is told that you were here.
  • One exception, stated plainly: version numbers come from a nightly cache file on this domain, but if that file is stale or missing an entry the page asks GitHub's public releases API for the tag. No cookies and no identifier are sent, and it is the only request that ever leaves this domain.
  • The page works with JavaScript disabled. Filtering the project list, opening a screenshot, and reading everything all work without it; scripting only adds URL memory and the lightbox.
  • Images ship as WebP with a PNG fallback, sized so a phone never downloads a desktop-sized screenshot.
  • A first visit costs about 300 KB including fonts and the hero screenshot. budget.sh in the site repository measures it, so the number in the footer can be checked.

If something here is wrong

If any of this fails for you — a contrast that does not hold up, a control you cannot reach, motion that slipped through — write to calstfrancis@disroot.org and I will fix it. A commitment nobody can report against is just a slogan.
